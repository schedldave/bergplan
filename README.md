# Bergplan – 10 Wanderungen zum Traunstein

Ein progressiver Wander-Trainingsplan: zehn Vortouren rund um Linz, Salzkammergut und Mostviertel, mit dem Traunstein als Ziel.

Live unter GitHub Pages.

## Aufbau

- `index.html` – Übersichtsseite mit allen Touren
- `styles.css` – Layout & Typo
- `.nojekyll` – Pages ohne Jekyll-Verarbeitung

## Lokale Vorschau

```bash
python3 -m http.server 8000
# dann http://localhost:8000
```

## Lizenz

Privater Plan. Keine Werbung, kein Tracking.
